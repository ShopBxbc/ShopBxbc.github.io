# BXBC – Bharatpur Xerox & Book Centre

A modern, responsive business website for **BXBC – Bharatpur Xerox & Book Centre** built using HTML, CSS and JavaScript.

---

## 🌟 Features

- 📄 Xerox & Photocopy Services
- 🖨️ Document Printing
- 📚 Notebooks
- ✏️ Stationery
- 🛡️ Lamination
- 📱 Mobile Recharge
- ⭐ Responsive Design
- 🚀 Fast Loading
- 📱 Progressive Web App (PWA)
- 💬 WhatsApp Integration
- 📞 Click-to-Call Support
- 📍 Google Maps Integration
- ✨ Smooth Animations
